CASPs must verify all entities using their platform. Verun's KYA score provides a machine-readable, auditable identity record for every AI agent before access is granted.

AML/CFT Obligations

Verun's 2-of-3 validator consensus acts as a distributed AML screening layer. Agents with scores below threshold are automatically blocked — no manual intervention required.
